iPad-LearnEverything

勇者斗恶龙: An iPad game for children to learn Chinese and for teacher to easily create questions

Main Features

  • Teacher can easily create their own question set for children just by using iPad
  • One-click share of question set to all children's iPad
  • Support three types of questions: Math question, Chinese/English translation, Picture/Word matching
  • Fun card matching game, support two players on the same screen!
  • More features developing...

To clone this project:

$ git clone https://github.com/gyfelton/iPad-LearnEverything.git
